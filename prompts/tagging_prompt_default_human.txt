Output JSON schema:
{{
  "tone": string,        // e.g., enthusiastic, calm
  "emotion": string,     // e.g., curious, excited
  "focus": string,       // e.g., conversation building, character description

Rules:

• Use [] for zero or more values, and "" or null for empties.
• Do *not* emit any extra fields or prose—just the JSON object.
